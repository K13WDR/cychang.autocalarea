AutoCalArea CAD 自動面積計算與標註工具 — 指令使用說明

適用: AutoCAD LWPOLYLINE 檔案: AutoCalAreaV1_Beta.lsp 測試對象: 封閉多邊形 (70=1)
工具簡介:自動辨識封閉 LWPOLYLINE 的幾何形狀,產生計算式、圖面標註與彙總表格。 支援矩形、梯形、三角形、扇形、弓形,以及複合圖形的自動分解(如矩形+弓形、扇形+凹口等)。

檔案版本選擇

本工具提供兩個編碼版本,請依你的系統環境選擇:

檔案名稱 編碼 適用環境
AutoCalAreaV1_Beta_Big5.lsp Big5 一般 Windows 繁體中文系統(預設環境,未啟用 UTF-8)
AutoCalAreaV1_Beta_UTF8.lsp UTF-8 Windows 11 已啟用「世界各國語言支援 Unicode UTF-8」Beta 功能者

如何判斷要挑哪一個?

  • 若你沒有特別設定過系統語言選項 → 選 Big5 版本
  • 若你在 Windows 11 的「地區設定 → 系統管理 → 變更系統地區設定」有勾選「使用 Unicode UTF-8 以提供全球語言支援」的 Beta 選項 → 選 UTF-8 版本
  • 若載入後命令列訊息或標註文字出現亂碼 → 代表選錯版本,改用另一個重試

載入方式

  1. 在 AutoCAD 命令列輸入 APPLOAD 或透過功能表 工具 → 載入應用程式
  2. 選取對應編碼的 .lsp 檔案
  3. 載入成功後命令列會顯示 成功載入 訊息
  4. 之後就可以使用 ACA、ACAO、ACAL、ACAR 等指令

指令總覽

完整指令 縮寫 功能
AutoCalArea ACA 執行主要面積計算與標註
AutoCalAreaOption ACAO 調整顯示參數設定
AutoCalAreaLabel ACAL 自訂子形狀編號前綴
AutoCalAreaReset ACAR 重置所有設定為預設值

ACA — 主計算指令

AutoCalArea 縮寫 ACA

選取多邊形 → 點選表格位置 → 自動產生圖面標註與計算表。

使用流程

  1. 在命令列輸入 ACA 並按下 Enter 啟動
  2. 選取一個或多個封閉 LWPOLYLINE(可框選多個一起處理)
  3. 於圖面上點選計算表的起始位置(左上角)

自動產生內容

圖面:子形狀編號圈、主編號圈、邊長標註、半徑 / 角度標註、切割輔助線、圓心標記、直角符號。

表格:每個圖形的編號、計算式、面積 (m²),最後附上總面積 (Total Area)。

形狀自動辨識

  • 矩形 / 梯形 / 三角形 / 完整扇形 → 單一公式直接輸出
  • 含弧段或多邊形 → 自動分解為子形狀並組合計算式
  • 子形狀類別:扇形 S / 弓形 G / 矩形 J / 三角形 A / 梯形 T(前綴可自訂)

防蠢機制

執行時會自動偵測並跳過不合法圖形,不會造成當機:

  • 線段重疊 多邊形有邊線來回重疊 → 命令列警告並跳過
  • 自我交結 任兩條邊彼此跨越(例如連續弧段造成自交)→ 命令列警告並跳過

ACAO — 參數設定

輸入 ACAO 後選擇要修改的選項。未修改的項目沿用上次設定。

選項 快捷 預設值 說明
文字高度 T 15 所有標註、編號、計算表的文字大小
單位比例 U 100 圖紙單位換算實際 m,例如 cm → 100
主編號行距 M 2.0 計算表主編號間的垂直行距倍數(以文字高度為基底)
子編號行距 S 1.8 同一主編號內多行計算式的行距倍數
符號比例 Y 0.8 圓心標記、直角符號相對於文字高度的大小比例
離開 X — 完成設定、返回繪圖環境

ACAL — 子編號前綴設定

輸入 ACAL 後可分別自訂五種子形狀的編號字母。目前設定值會顯示在命令列,按 Enter 可保留原值。

形狀 快捷 預設前綴 編號範例
扇形 S S S1, S2, S3...
弓形 G G G1, G2, G3...
矩形 R J J1, J2, J3...
三角形 A A A1, A2, A3...
梯形 T T T1, T2, T3...
離開 X — 完成設定

ACAR — 重置

輸入 ACAR 立即將所有參數(文字高度、比例、行距、符號比例、前綴)全部還原為預設值。不需確認、不會影響已繪製的圖面內容。

自動建立的圖層

指令執行時會自動建立以下圖層(若已存在則沿用)。你可以自行調整顯示、隱藏或列印屬性。

圖層名稱 顏色 線型 用途
AREA-文字 3 (綠) 實線 計算表文字、編號
AREA-數值 6 (紫) 實線 邊長、半徑、角度標註
AREA-編號 3 (綠) 實線 主編號圈、子編號圈
AREA-切割線 8 (灰) 實線 三角剖分對角線、弓形弦
AREA-輔助線 9 (淺灰) 虛線 圓心連接線、三角形高線
AREA-符號 9 (淺灰) 實線 圓心十字、直角符號

測試重點

請協助測試以下情境,並記錄結果。

基本形狀

  • 單一矩形、梯形、三角形
  • 半圓、完整扇形
  • 傾斜角度的矩形 / 梯形(30°、45°、60°、> 45°)

複合圖形

  • 矩形 + 半圓 / 弓形(外凸或內凹)
  • 梯形扣除弓形
  • 五邊形以上的不規則多邊形
  • 含多個弧段的組合

極端情境

  • 大於 180° 的扇形(例如吃豆人形狀)
  • 內凹弓形 + 矩形組合
  • 極小或退化的子三角形(確認是否被自動過濾)

設定變更

  • 修改文字高度、比例、符號比例,觀察是否即時生效
  • 自訂子編號前綴後,確認圖面與計算表都正確套用
  • 執行 ACAR 後確認所有設定回到預設

防蠢機制

  • 刻意繪製邊線重疊的多邊形 → 應跳過並顯示警告
  • 刻意繪製自交多邊形 → 應跳過並顯示警告
  • 確認程式不會因此當機、也不會污染其他正常圖形的處理

回報格式

發現問題或建議時,請提供以下資訊以便快速定位:

  • 圖形截圖:包含主編號、子編號、所有標註、計算表
  • 命令列訊息:特別是 [DEBUG] 與 [ACA 警告] 等輸出
  • 預期 vs 實際:簡述你預期看到什麼、實際看到什麼
  • 測試環境:AutoCAD 版本、作業系統
